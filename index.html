Scrmbled

Repair the machine

Figure out what the machine does to words, then fix a new one. You get 6 tries.

How to play

Goal: Discover the transformation rule, then repair a broken word.

Phase 1 — Find the rule

You are shown two word transformations:

SILVER → REVLIS
RANDOM → MODNAR

Type what rule changes the first word into the second.

Examples of valid answers:

  • reverse
  • swap first and last
  • rotate left

Phase 2 — Repair the machine

After identifying the rule, a corrupted word appears:

ELZZUP

Apply the same rule to repair it and type the correct word.

Rules

  • You get 6 total guesses across both phases
  • The timer starts automatically when the puzzle loads
  • Only one puzzle per day
  • Spelling matters

Tip: The examples always use the exact same rule as the final repair.
